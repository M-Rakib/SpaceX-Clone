# SpaceX Website Clone

A responsive **SpaceX website UI clone** built using **HTML, CSS, and JavaScript**, inspired by the official SpaceX website.  
This project focuses on modern layout design, full-screen hero sections, animations, and a clean multi-page structure using static image assets.

## Pages Included
- Home page (`index.html`)
- Falcon 9 page (`falcon9.html`)
- Falcon Heavy page (`falcon-heavy.html`)
- Dragon page (`dragon.html`)

## Features
- Responsive navigation bar with mobile hamburger menu
- Full-screen hero sections with launch imagery
- Smooth transitions and animations
- Animated statistics counters
- Multi-page navigation structure
- Mobile-first and responsive design

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts
